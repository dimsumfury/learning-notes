# python terms go here

Statement
part of the program which executes something
example:
if name == "Anna":
    print("Hi!")
    number = 2

Block
group of consecutive statements that are at the same level in the structure of the program
example:
if age > 17:
    # beginning of the conditional block
    print("You are of age!")
    age = age + 1
    print("You are now one year older...")
    # end of the conditional block

print("This here belongs to another block")

Expression
code that results in a determined data type
example:
2+4+3=9                  integer                   int
"abc" + "de" = "abcde"   string                    str
11/2=5.5                 floating point number     float
2*5>9  true              boolean value             bool

Function
executes some functionality  
example:
print("this is an argument") - the argument is what is printed

Data type
characteristics of any value present in the program
example:
string (str) or integer (int) etc.
name = "Anna"
result = 100
use type to find out what the data type is
print(type("Anna"))
print(type(100))
output:
<class 'str'>
<class 'int'>

Syntax
determines how the code of a program should be written
If the rules of the language are not followed, there will be an error

Debugging
If the syntax of the program is correct but the program still doesn't function as intended, there is a bug in the program
